Learn HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

In this course, you will learn basic HTML elements and their attributes. HTML is the beginning of everything you need to know to create engaging web pages!

WHAT YOU'LL LEARN

Day 1
Getting Started		 On this day, you will learn about all of the basic things you need to start coding for your HTML Website. You will be learning about (a) different HTML editors that is being used to write HTML codes; (b) different HTML sections in an HTML Website (c) two types of HTML element and; (d) anatomy of an HTML element.
Day 2
Texts		 Texts are one of the most used element in a webpage. On this day, you will learn different text and text formatting techniques. This section will introduce you to headers (from 1 to 6), the paragraph tag, and text formatting tags including (but not limited to) italic, bold, subscript and superscript tags.
Day 3
Lists		 From list of continents up to your grocery lists, Lists are very important part of Web Pages. On this day, you will learn different types of lists - unordered, ordered, and description lists.
Day 4
Images		 Images make your design lively and appealing. Additionally, images give depth and context to a description or story and provide a much more immersive experience than writing alone. That is why on this day, we will make sure that you know how to add images to a website.
Day 5
Links		 Links are a vital part of HTML and the very reason why the WWW exists. In general, these links, together with browsers' functionalities, allow users to move from one document to another in a very direct way. That is why in this section, you will learn about HTML links.
Day 6
Table		 HTML tables are used for displaying data that make sense in spreadsheet software. They consist of rows and columns and are often used on websites for the effective displaying of tabular data. In this section, you will learn how to create tables and some of its basic styling to create a table with a style.
Day 7
Forms		 On the internet, the most-used tag is form tag; without form tags, the internet would be a read-only repository of boring documentation. On this section, you will learn how to create forms and differnt types of inputs such as texts, radio, checkboxes and submit button.
