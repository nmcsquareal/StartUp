HTML Links

Links are found in nearly all web pages. It is a vital part of HTML. In general, these links, toghether with browsers' functionalities, allow users to move from one document to another in a very direct way.

HTML Links Syntax

The HTML <a> tag defines a hyperlink. a stands for anchor, so this tag is sometimes being called as an anchor tag.

Similar to the <img> tag, the <a> tag also have attributes. The most important attribute, however, is the href attribute. An example of an HTML link with the href attribute is presented below.

<a href="https://www.youtube.com/"> Go to Youtube to watch Educational videos </a>

Go to Youtube to watch Educational videos

The anchor tag above has an href (stands for hypertext reference) attribute. The href has the youtube link as its attribute. That means the inside of the anchor tag (the text: Go to Youtube to watch Educational videos) will become a hyperlink. That means that clicking the text will link you to Youtube.

Note that by default, enclosing an element in an anchor tag will change its color to purple. This will also add an underline to the element if it is a text.

The Target Attribute

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

The target attribute specifies where to open the linked document. The table below shows the meaning of each possible values of the target attribute.

_self This is the default value of the target attribute. It opens the document in the same window/tab as it was clicked.
_blank This opens the document in a new window or tab.
_parent This opens the document in the parent frame.
_top This opens the document in the full body of the window.

Example:

<a href="https://www.youtube.com/" target="_blank"> Go to Youtube to watch Educational videos </a>

HTML Links in an Image

HTML Links are not only for texts. You can also enclosed an image in an anchor tag to make it a link.

Example:

<p>This image is a link</p>

<a href="https://www.youtube.com/">

<img src="img/codepic.jpg " style="max-width:400px">

</a>

The image below is a link (You can try clicking the image)

As you can see in the example, the <img> is enclosed in an anchor tag. This will make the image clickable and will link you to the youtube page. Note that unlike on texts, this will not change the color of the image nor add an underline.

Link Titles

The last attribute that you will learn in this section is Link Titles. The title attribute specifies extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element. Below is an example of a link showing its title.

<a href="https://www.youtube.com/" title="Go to Youtube"> Youtube </a>

Youtube

"Try hovering your mouse on the youtube link above"

If you include a title on a link tag, the value will show when your mouse hover through the link.
