HTML Tables

A table is a structured set of data made up of rows and columns (tabular data). A table allows you to quickly and easily look up values that indicate some kind of connection between different types of data, for example a person and their age, or a day of the week, or the timetable for a local swimming pool.

HTML Table Syntax

A table is declared using the <table> tag. Since a table contains rows and columns, you will also need a syntax to declare them. In HTML, we start with rows by declaring the <tr> tag. Then, we declare the <td> tag for every cell for that first row.

A table below summarizes the common HTML tags being used to create a table.

<table> Define a table
<tr> Define a table row
<td> Define a table cell
<th> Define a table header cell. This is commonly being used at the uppermost part for the title or header of each column.

Example:

<!DOCTYPE html>

<html>

<head>


</head>

<body>

<table>

<tr>

<th> Name </th>

<th> Sex </th>

<th> Age </th>

</tr>

<tr>

<td> Neal </td>

<td> Male </td>

<td> 23 </td>

</tr>

<tr>

<td> Lance </td>

<td> Male </td>

<td> 20 </td>

</tr>

<tr>

<td> Victoria </td>

<td> Female </td>

<td> 21 </td>

</tr>

</table>

</body>

</html>

Name Sex Age
Neal Male 23
Lance Male 20
Victoria Female 21

As you can see, the data were presented systematically into a table. The <th> tag content (uppermost part) were presented in bold as the header. There are four rows from the four <tr> and the cells were encoded using <td>.

HTML Table with Border

You can add border to your HTML table. By adding an attribute to the table tag from the previous example, we can add border to the whole table.

< table style="border: 1px solid black;" >

Name Sex Age
Neal Male 23
Lance Male 20
Victoria Female 21

The border styling adds a border to the table (but not to the cells since the styling is only added to only the border. The values of the border requires the width (such as width in pixels), style (such as solid, dotted, dashed or double) and color. You will learn more about these styling in CSS.

HTML Cells with Border

You may also add border to every cells in the table. To do this, you just need to add the styling to the <td> tag.

<table style="border: 1px solid black;">

<tr>

<th style="border: 1px solid black;"> Name </th>

<th style="border: 1px solid black;"> Sex </th>

<th style="border: 1px solid black;"> Age </th>

</tr>

<tr>

<td style="border: 1px solid black;"> Neal </td>

<td style="border: 1px solid black;"> Male </td>

<td style="border: 1px solid black;"> 23 </td>

</tr>

<tr>

<td style="border: 1px solid black;"> Lance </td>

<td style="border: 1px solid black;"> Male </td>

<td style="border: 1px solid black;"> 20 </td>

</tr>

<tr>

<td style="border: 1px solid black;"> Victoria </td>

<td style="border: 1px solid black;"> Female </td>

<td style="border: 1px solid black;"> 21 </td>

</tr>

</table>

Name Sex Age
Neal Male 23
Lance Male 20
Victoria Female 21

As you can see, you now have border for every cell in the HTML table. However, sometimes it looks ugly because of the spacing between border cells. That because of default HTML 5 table has a spacing for border cells.You may add a border-spacing attribute to the table tag and set this to 0 to remove this.

<table style="border: 1px solid black; border-spacing:0;">
