Images

Images are very important. Images can improve the design and the appearance of a web page. It can also help create a connection with the texts and can clarify the information presented in a layout by adding meaning and evoking associations. Adding image on its own, however, is not enough to make it powerful.

That is why for this section, you will learn how to add images as well as its important attributes.

HTML Images Syntax

The HTML <img> tag is used to embed an image in a web page.

Note that the <img> tag is a self-closing tag. That means that it does not need a closing tag.

Below is an example of an image tag with its attributes.

<!DOCTYPE html>

<html>

<head>


</head>

<body>

<img src="codepic.jpg" alt="Picture of a code" style="max-width: 500px;">

</body>

</html>


The <img> tag on the image above has the attributes src, alt, and style. Each attributes also have their corresponding values. Below is a table of these attributes and their meaning.


src The src attribute specifies the path to the image. It can be in the form of a link or a file path.
alt The alt attribute specifies an alternate text for the image. This will be presented when the image cannot be displayed (because of slow connection, an error in the src attribute, or if the user uses a screen reader).
style The style attribute specifies some styling on the image. In the example, the max-width of the image is 500px. This means that html will instantly set the maximum image to the set value.

Images in another Folder

If you have an image from another folder, you must include the folder name in the src attribute.

Example: <img src="images/codepic.jpg" alt="Picture of a Code" style="width:620px; height:380px;">

In the example above, the image codepic.jpg is in a folder named images.
