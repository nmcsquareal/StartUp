Getting Started

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

To learn about HTML, we will start by knowing about text editors, basic HTML sections and the types of HTML tags.


Text Editors

In order to start writing HTML codes, you will be needing a text editor - some software where you can write the code in the first place. You may use professional HTML editors such as Visual Studio Code (VS Code) but simple text editor like Notepad will do.

Below are some images of text editors (Notepad for simple text editor and Visual Studio Code for professional one). You may click on the image to go to their website so you may download them for free.


Basic HTML Sections

Presented below are the basic sections in HTML.

<!DOCTYPE html>

<html>

<head>


</head>

<body>

</body>

</html>

The first line is what we call a Document Type declaration (<!DOCTYPE html>). This declares the type of document - and in this case, it is an HTML document. Every HTML document should start with this tag.

Next, we add an HTML tag. Notice that you can see two version of this tag on the second and last line. The first version (<html>) is an opening tag and the second version in the 11th line (</html>) is the closing tag. This means that everything inside the opening and closing tag of the html element is part of it.

This is the usual pattern for a lot of html elements. Presented below is a simplified way of writing HTML elements. The example is a paragraph tag (<p> </p>). You will learn about paragraph tags soon.

You may also notice that this does not work on <!DOCTYPE html>. This is because the document declaration tag is a self-closing tag (also called as Empty Tag). This means that it does not need a closing tag. Other examples of self-closing tags are break tag (<br>) and horizontal rule tag <hr>. You will learn more about them soon.

Embedded in the html tag are two parts of the HTML Website, the head tag and the body tag.

The head tag (<head>) contains links and meta information about the HTML page. Mostly these links and meta information does not directly display any HTML element but they are important for the functionality of the website. A good example is when you add a CSS link to the head section to add styling to your website.

The body tag (<body>) defines the document's body. The body element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. This is the section where you are going to work much often as this area works on something that can be seen by your website's viewer.
