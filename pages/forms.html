HTML Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing. To declare a form element, we need the <form> tag. Inside the form tag, we can add input elements such as text fields, checkboxes, radio buttons, submit buttons, etc.

The Input element

The HTML <input> element is the most used form element. It can be displayed in many ways depending on the type attribute.

Type Description
<input type="text"> Displays a single-line text input field
<input type="radio"> Displays a radio button (for selecting one of many choices)
<input type="checkbox"> Displays a checkbox (for selecting zero or more of many choices)
<input type="submit"> Displays a submit button (for submitting the form)
<input type="button"> Displays a clickable button

Text Fields

The <input type="text"> tag defines a single-line input field for text input. This works for names, address, birthplace, or anything that requires just texts.

<!DOCTYPE html>

<html>

<head>


</head>

<body>

<form>

<label for="fname"> First Name: </label><

<input type="text" id="fname" name="fname"><br>

<label for="lname"> Last Name: </label><

<input type="text" id="lname" name="lname"><br>

</form>

</body>

</html>

The Label element

Notice that before the input, we added a label element. The <label> element also help users who have difficulty clicking on very small regions (such as radio buttons or checkboxes) - because when the user clicks the text within the < label> element, it toggles the radio button/checkbox.

The <for> attribute of the <label> tag should be equal to the id attribute of the <input> element to bind them together.

Radio Buttons

The <input type="radio"> defines a radio button. Radio buttons let a user select ONE of a limited number of choices.

<form>

<input type="radio" id="adobo" name="fav_food">

<label for="adobo"> Adobo </label> <br>

<input type="radio" id="sinigang" name="fav_food">

<label for="sinigang"> Sinigang </label> <br>

<input type="radio" id="paksiw" name="fav_food">

<label for="pakswi"> Paksiw </label> <br>

</form>

Checkboxes

The <input type="checkbox"> defines a checkbox. Checkboxes let a user select ZERO or MORE options of a limited number of choices.

<form>

<input type="checkbox" name="arianagrande" value="Ariana Grande"><br>

<label for="arianagrande"> Ariana Grande </label><

<input type="checkbox" name="tswift" value="Taylor Swift"><br>

<label for="tswift"> Taylor Swift </label><

<input type="checkbox" name="ladygaga" value="Lady Gaga"><br>

<label for="ladygaga"> Lady Gaga </label><

</form>

The Submit Button

The <input type="submit"> defines a button for submitting the form data to a form-handler. The form-handler is typically a file on the server with a script for processing input data.The form-handler is specified in the form's action attribute.

<form action="/action_page.php">

<label for="fname"> First Name: </label><

<input type="text" id="fname" name="fname" value="John"><br>

<label for="lname"> Last Name: </label><

<input type="text" id="lname" name="lname" value="Doe"><br>

<input type="submit" value="Submit">

</form>



The Name attribute

Notice that each input field must have a name attribute to be submitted.

If the name attribute is omitted, the value of the input field will not be sent at all.
