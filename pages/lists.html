HTML Lists

Lists are used to group together related pieces of information so they are clearly associated with each other and easy to read. In HTML, you can actually create lists. However, there are three types of lists - unordered, ordered, and description lists.

Unordered Lists

An unordered list starts with <ul> tag and ends with </ul>. The list items should then be enclosed in <li> tag.

Below is an example of an unordered list in HTML.

<!DOCTYPE html>

<html>

<head>


</head>

<body>

<h1> Grocery List </h1>

<ul>

<li> Coffee <li>

<li> Milk <li>

<li> White Sugar <li>

<li> Flour <li>

</ul>

</body>

</html>

Grocery List

  • Coffee
  • Milk
  • White Sugar
  • Flour

Ordered Lists

An ordered list starts with <ol> tag and ends with </ol>. Similar to an unordered list, list items should then be enclosed in <li> tag.

Below is an example of an ordered list in HTML.

<!DOCTYPE html>

<html>

<head>


</head>

<body>

<h1> Largest Country in the World by area: </h1>

<ol>

<li> Russia <li>

<li> Canada <li>

<li> China <li>

<li> USA <li>

<li> Brazil <li>

</ol>

</body>

</html>

Largest Country in the world by area:

  1. Russia
  2. Canada
  3. China
  4. USA
  5. Brazil

Description Lists

A description list is a list of terms, with a description of each term (similar to a dictionary).

A description list starts with <dl> tag and ends with </dl>. The <dt> tag defines the term while the <dd> tag describes each term.

Below is an example of a description list in HTML.

<!DOCTYPE html>

<html>

<head>


</head>

<body>

<h1> Terminologies: </h1>

<dl>

<dt> Sun </dt>

<dd> central star in our solar system. </dd>

<dt> Earth </dt>

<dd> livable planet. </dd>

<dt> Moon </dt>

<dd> natural satellite of Earth. </dd>

</dl>

</body>

</html>

Terminologies:

Sun
central star in our solar system.
Earth
livable planet.
Moon
natural satellite of Earth.
